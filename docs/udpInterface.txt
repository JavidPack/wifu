Requirements for the UDP Interface

1. Listening module, which calls a callback with data
2. Sending interface: ability to send data to a listening module (send to an address, port)
3. Do we want to send data, packet objects?
4. Do we want to receive data, packet objects?

How to map Wifu socket to actual UDP socket(s) and vica-versa.
Send will need to map Wifu socket to address, port, and socket.
Receive will need to map UDP socket to address, port, and Wifu Socket
